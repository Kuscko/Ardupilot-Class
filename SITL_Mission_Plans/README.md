# SITL Mission Plans — Agent Onboarding

## Purpose
Create, run, and document mission plans in SITL (Software-In-The-Loop) for ArduPilot (Plane 4.5.7). Capture all test scenarios, parameter explorations, and results during onboarding.

- [ ] Set up SITL environment (see onboarding guide Week 1)

### 2. Mission Planning
- [ ] Create and run example mission plans (takeoff, waypoint, RTL, land)
- [ ] Experiment with flight modes and parameter changes (TECS, failsafes, PIDs, serial/hardware)
- [ ] Document each test scenario and its results

### 3. Troubleshooting
- [ ] Record any issues encountered during SITL testing
- [ ] Add solutions for common SITL problems (see onboarding guide Appendix B)
