# Lua Scripts — Agent Onboarding

## Purpose
Learn, create, and document Lua scripts for ArduPilot (Plane 4.5.7). Capture all experiments, findings, and best practices during onboarding.

- [ ] Run and test example Lua scripts (see onboarding guide Day 10)

### 2. Script Development

- [ ] Experiment with reading/writing parameters, controlling outputs, and sending messages
- [ ] Document all scripts and their results

### 3. Troubleshooting
- [ ] Record any issues with script loading or execution
- [ ] Add solutions for common Lua scripting problems
